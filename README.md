# Bang Wu academic homepage draft

This is a static GitHub Pages homepage draft for Dr. Bang Wu at RMIT University.

It contains:

- `index.html` — the homepage content
- `style.css` — the visual design
- `.nojekyll` — tells GitHub Pages to publish the files exactly as written

## Important verification notes

This page was drafted from public sources only. Before publishing, verify the biography, publication list, links, current title, supervision topics, and any claims about research interests. Replace the initials avatar with an approved headshot if available.

## Deploy on GitHub Pages using only the browser

1. Sign in to GitHub.
2. Create a repository named `YOUR-GITHUB-USERNAME.github.io`.
3. Make the repository public unless your account plan supports private GitHub Pages.
4. Upload `index.html`, `style.css`, and `.nojekyll` to the repository root.
5. Commit the files.
6. Open repository `Settings`.
7. Open `Pages` in the left sidebar.
8. Under `Build and deployment`, choose `Deploy from a branch`.
9. Select branch `main` and folder `/ (root)`.
10. Click `Save`.
11. Visit `https://YOUR-GITHUB-USERNAME.github.io` after GitHub finishes deployment.
